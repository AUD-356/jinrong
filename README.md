渗透测试练习使用公告
为规范网络安全学习行为，为广大网络安全爱好者、学习者提供合法、安全的渗透测试练习环境，特此发布本使用公告，所有使用本练习场地、练习资源的人员，必须严格遵守以下规范：
1. 本平台/场地仅用于个人网络安全技术学习、合法渗透测试实操练习、安全技能实训，所有练习内容均为模拟漏洞环境、虚拟靶机、仿真业务系统，仅限授权内部学习使用。
2. 所有使用者必须遵守《中华人民共和国网络安全法》《中华人民共和国数据安全法》《中华人民共和国个人信息保护法》等相关法律法规，严禁利用本平台技术、工具、思路对外网真实网站、服务器、设备、公众网络资源进行未授权渗透、攻击、扫描、入侵等违法行为。
3. 使用者仅限在专属虚拟练习环境内开展漏洞挖掘、代码审计、权限测试、漏洞修复等合规练习操作，禁止超出练习范围进行违规操作、恶意测试、破坏环境稳定性的行为。
4. 严禁利用本练习环境从事任何非法牟利、网络攻击、网络破坏、窃取数据、入侵他人系统等违法违规活动，一经发现，将立即封禁使用权限，保留截图、日志等证据，并移交相关部门追究法律责任。
5. 本练习环境所有操作日志全程留存，所有使用者操作行为可追溯，请自觉规范自身操作行为，坚守合法合规的网络安全学习底线。
